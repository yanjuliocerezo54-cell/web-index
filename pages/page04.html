REDES INFORMÁTICAS

Fundamentos, Modelos, Protocolos y Estructura

1. INTRODUCCIÓN

En la actualidad, las redes informáticas constituyen uno de los pilares fundamentales de la tecnología moderna. Gracias a ellas es posible la comunicación instantánea entre personas, empresas e instituciones en cualquier parte del mundo. Las redes permiten compartir información, recursos y servicios de manera rápida, segura y eficiente.

Desde el uso del internet en nuestros hogares hasta los grandes centros de datos empresariales, todo funciona mediante sistemas de interconexión organizados bajo modelos, protocolos y estructuras específicas.

El estudio de las redes es esencial en la Ingeniería de Sistemas, ya que permiten comprender cómo viaja la información, cómo se protege y cómo se administra dentro de una infraestructura tecnológica.

2. ¿QUÉ ES UNA RED INFORMÁTICA?

Una red informática es el conjunto de dispositivos interconectados entre sí que comparten información mediante medios físicos o inalámbricos utilizando reglas de comunicación llamadas protocolos.

El objetivo principal de una red es compartir datos, compartir recursos (impresoras, archivos, internet), facilitar la comunicación y optimizar procesos empresariales.

3. MODELOS DE RED: OSI Y TCP/IP

MODELO OSI (7 CAPAS)

El modelo OSI divide la comunicación en siete capas,el modelo OSI organiza el proceso de transmisión de datos en siete capas, permitiendo comprender de manera estructurada cada etapa de la comunicación, desde la transmisión física de la información hasta la interacción con el usuario:

  • Física: Transmite bits a través de cables o señales.
  • Enlace de Datos: Controla errores y utiliza direcciones MAC.
  • Red: Maneja direcciones IP y enrutamiento.
  • Transporte: Controla la entrega de datos (TCP/UDP).
  • Sesión: Establece y mantiene la conexión.
  • Presentación: Traduce y cifra la información.
  • Aplicación: Interactúa directamente con el usuario.

    • Este modelo es teórico y sirve como referencia para estudiar redes

MODELO TCP/IP (4 CAPAS)

Por su parte, el modelo TCP/IP simplifica este proceso en cuatro capas y constituye la base del funcionamiento de Internet, definiendo cómo los datos se envían, direccionan y reciben entre computadoras.:

  • Acceso a la Red
  • Internet
  • Transporte
  • Aplicación

    • Es más simple que el OSI y combina varias capas en una sola.

4. PROTOCOLOS DE RED

Es un conjunto de reglas, normas y procedimientos correctos que permiten que dos o más dispositivos se comuniquen entre sí dentro de una red.

  • IP → Identifica dispositivos.
  • TCP → Garantiza entrega confiable.
  • UDP → Más rápido pero sin confirmación.
  • HTTP / HTTPS → Navegación web.
  • FTP → Transferencia de archivos.
  • DNS → Traduce nombres a direcciones IP.
  • SMTP, POP3, IMAP → Correo electrónico.

    • Sin protocolos, la comunicación sería imposible

5. TOPOLOGÍAS DE RED

Las topologías de red son la forma o estructura en que están organizados y conectados los dispositivos dentro de una red informática.

  • Topología en Estrella: Todos conectados a un switch central.
  • Topología en Bus: Todos comparten un cable principal.
  • Topología en Anillo: Conectados en forma circular.
  • Topología en Malla: Conexión múltiple entre dispositivos.

6. DISPOSITIVOS DE RED

Los dispositivos de red son los equipos físicos que permiten la conexión, comunicación y transmisión de datos entre computadoras y otros dispositivos dentro de una red informática.

En otras palabras, son los aparatos que hacen posible que la información viaje de un lugar a otro.

  • Router
  • Switch
  • Módem
  • Access Point
  • Servidor

7. CABLEADO UTP

El cable UTP (Unshielded Twisted Pair) es el más utilizado en redes LAN. Contiene 8 hilos trenzados en pares que reducen interferencias, es económico, flexible y soporta altas velocidades.

  • Categorías: Cat5e, Cat6, Cat6a.
  • Estándares: T568A y T568B.
  • Tipos: Cable directo y cable cruzado.

8. SOFTWARE Y SIMULADORES

En redes no solo se trabaja con hardware, también con software especializado para configurar, analizar y simular infraestructuras de red.

  • Sistemas Operativos de Red: Windows Server, Linux Server.
  • Simuladores: Cisco Packet Tracer, GNS3.
  • Analizadores: Wireshark.
